# HomePage Design Analysis

This document is now a compact Flutter-native Home analysis. The old web
blueprint was removed with the Flutter-only cleanup.

## Home Role

- Establish global app chrome behavior.
- Set the neutral dark surface system.
- Demonstrate the primary brand treatment.
- Provide entry points into core modules without expanding bottom navigation.

## Design Requirements

- Trust-first, finance-oriented hierarchy.
- Clear balance/portfolio treatment with masked-value support where needed.
- Scannable market/action modules.
- No hype, FOMO, or casino-style presentation.
- Native safe-area behavior on real devices.
